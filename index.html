☁️ Google Drive সেটআপ

🔑 Google Client ID কী?
এটা একটা ফ্রি কোড — একবার তৈরি করলেই হয়। এই কোডটা দিলেই আপনার অ্যাপ Google Drive-এ ব্যাকআপ সেভ করতে পারবে।
📋 কিভাবে Client ID পাবেন (৫ মিনিট):
১. console.cloud.google.com খুলুন
২. "New Project" → নাম দিন "MarketApp" → Create
৩. বাম মেনু → APIs & Services → OAuth consent screen
    → External → Create → App name দিন → Save
৪. Credentials → Create Credentials → OAuth Client ID
    → Application type: Web application
    → Authorized JavaScript origins-এ আপনার সাইটের URL দিন
       (লোকাল ফাইল হলে: null বা http://localhost)
৫. Create → Client ID কপি করুন → নিচে পেস্ট করুন
🔒 এই Client ID শুধু আপনার ডিভাইসে সেভ থাকে। কোথাও পাঠানো হয় না।

📧 ইমেইলে ব্যাকআপ পাঠান

💡 নিচে আপনার ইমেইল ঠিকানা দিন। ব্যাকআপ ডেটা কপি করে Gmail / Outlook এ পেস্ট করে পাঠাতে পারবেন, অথবা mailto লিংক ব্যবহার করে সরাসরি ইমেইল অ্যাপ খুলতে পারবেন।
